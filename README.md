# Last Minute Life Saver

## Project Name - FocusForge AI
## Problem Statement

Students and professionals often struggle with procrastination, poor time management, and creating realistic schedules around their daily commitments. Existing task managers require manual planning and do not adapt to individual lifestyles.

## Solution Overview

Last Minute Life Saver is an AI-powered productivity assistant that helps users organize their work efficiently. Users describe their schedule, commitments, and goals in natural language, and the application automatically generates prioritized tasks along with a realistic timetable.

Using Google Gemini, the system analyzes user requirements and creates a personalized action plan that integrates with task management and scheduling features.

## Key Features

* AI-powered task generation using Google Gemini
* Personalized timetable creation
* Priority-based task management
* Calendar scheduling and visualization
* Progress tracking
* Gamification through points, streaks, and badges
* Lifestyle-aware planning

## How It Works

1. User provides lifestyle information and goals.
2. Gemini analyzes the user's requirements.
3. AI generates prioritized tasks.
4. Tasks are automatically added to the planner.
5. Tasks are scheduled with recommended start and end times.
6. Progress is tracked through points and streaks.
